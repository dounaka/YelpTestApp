Yelp Assignment

Using the Yelp Fusion API (https://www.yelp.com/developers/documentation/v3), develop a native app that would fetch and display results of local businesses.

Requirements for this assignment are listed below:

App must be developed natively, and you may use any third-party libraries for this assignment
  • Search by business name
  • Search by address/city/postal code
  • Search by cuisine type
  • Sorting of results by distance and ratings
  • Easy to use UI
    • Detailed view of a result should include the following:
  • Name and photo of business
  • Categories
  • Hours of operation
  • Address and contact information
  • Ratings and snippet
  • Deals
  • Try to be creative and add any other feature not mentioned above. This will be counted as a bonus.
  • Code must be written in Swift 3+ for iOS or Java 7.0+ for Android
  • Code must compile and run on latest XCode and Android Studio with iOS 10.3+ and Android 5.0+
  • Upload code to cloud and send link to Jeremie.racine@bell.ca, James.Laurenstin@bell.ca, and Sami.lassed@bell.ca.
  • Explain briefly if any new features or extra work was completed
Source code : Repo Github

App development - Extra work and explanations

App "Hungry" shows the usage of
  • Rest API Volley
  • Data load with pagination ( loading resto by page of 20)
  • Android Jetpack / architecture component : LiveData and ViewModel
  • Fragment / Activity in a master/detail display
  • Support Tablet with a Master/Detail model
  • Custom View : SearchView, BusinessDetailView...
  • Managing the permission to get User Location - Request location permission compatible before and after Android M
  • App with material design style : put a theme that match the food industries - colors red and light gray
  • ViewModel with an activity lifecycle - allows communication between Activity & Fragment using livedata and observer
  • Pattern used : Factory, Singleton, ViewModel, Observer

Missing or bugs ...

  • Sorting by distance/rating doesn't seem to work using the API - reference : github issue
  • Search by cuisine not implemented
  • Deals not implemented
  • Bug on orientation change, state of the view is not restored
